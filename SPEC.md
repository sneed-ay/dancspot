# おどるーむ (Odoroom) - 社交ダンス教室まとめサイト

## 概要
全国の社交ダンス教室を都道府県・エリア別に探せるまとめサイト。
パネマジ掲示板（panemaji.com）の階層構造を参考に、地域→教室の導線で設計する。

## サイト構成

### ページ階層
1. **トップページ** `/`
   - 地方ブロック（北海道・東北・関東・中部・近畿・中国・四国・九州沖縄）
   - 各ブロック内に都道府県リンク
   - サイト説明・検索ボックス

2. **都道府県ページ** `/prefecture/tokyo` etc.
   - その都道府県の教室一覧
   - エリア（市区町村）でフィルタ可能
   - 教室カード: 教室名・住所・対応種目・料金帯

3. **教室詳細ページ** `/studio/{id}`
   - 教室名
   - 住所・電話番号・公式サイトURL
   - Google Maps埋め込み
   - 対応種目（ワルツ、タンゴ、ルンバ、チャチャチャ、サンバ等）
   - レッスン形態（グループ/個人/体験レッスン）
   - 料金情報
   - 営業時間

### 口コミ機能
初期版では実装しない（将来的に追加予定）

## 技術スタック
- **フレームワーク:** Next.js (App Router) + TypeScript
- **スタイリング:** Tailwind CSS
- **データ:** 初期はJSONファイル（将来的にDB移行可能な設計）
- **ホスティング:** Vercel（予定）
- **SEO:** 静的生成（SSG）で全ページを事前レンダリング

## デザイン方針
- モバイルファースト
- 清潔感のあるデザイン（白基調 + アクセントカラー）
- 日本語UI
- アクセントカラー: 上品な紫 or 深い青（社交ダンスの優雅さ）

## データ構造

```typescript
interface Studio {
  id: string;
  name: string;
  prefecture: string; // 都道府県コード e.g. "tokyo"
  prefectureName: string; // 東京都
  city: string; // 市区町村
  address: string; // 全住所
  phone?: string;
  website?: string;
  googleMapsUrl?: string;
  lat?: number;
  lng?: number;
  danceStyles: string[]; // ["ワルツ", "タンゴ", ...]
  lessonTypes: string[]; // ["グループ", "個人", "体験"]
  priceRange?: string; // "月額5,000円〜"
  hours?: string;
  description?: string;
}
```

## 初期データ
- Google Places API / Google Maps から教室情報を収集するスクリプトを用意
- まずは主要都市（東京、大阪、名古屋、福岡）のデータで開発
- 47都道府県の定義はハードコード

## 都道府県データ
47都道府県を以下の地方ブロックで分類:
- 北海道: 北海道
- 東北: 青森, 岩手, 宮城, 秋田, 山形, 福島
- 関東: 茨城, 栃木, 群馬, 埼玉, 千葉, 東京, 神奈川
- 中部: 新潟, 富山, 石川, 福井, 山梨, 長野, 岐阜, 静岡, 愛知
- 近畿: 三重, 滋賀, 京都, 大阪, 兵庫, 奈良, 和歌山
- 中国: 鳥取, 島根, 岡山, 広島, 山口
- 四国: 徳島, 香川, 愛媛, 高知
- 九州沖縄: 福岡, 佐賀, 長崎, 熊本, 大分, 宮崎, 鹿児島, 沖縄
